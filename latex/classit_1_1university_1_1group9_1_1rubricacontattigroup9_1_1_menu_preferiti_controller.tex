Gestisce l\textquotesingle{}apertura di una finestra popup per la selezione di un contatto. 

Questo metodo carica un file FXML per il popup, crea una nuova finestra ({\ttfamily Stage})

La finestra popup blocca l\textquotesingle{}interazione con altre finestre finché non viene chiusa.

\begin{DoxyPrecond}{Precondition}
Il file FXML "{}\+Seleziona\+Contatti\+Da\+Rubrica.\+fxml"{} deve essere presente e valido. 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
Una nuova finestra popup viene aperta, e l\textquotesingle{}utente non può interagire con altre finestre fino alla chiusura del popup.
\end{DoxyPostcond}
