\doxysubsection{Detailed Description}
Questa classe contiene il metodo per la validazione di un numero di telefono. 

La classe fornisce un metodo statico per verificare se un numero di telefono è valido, secondo un pattern regolare. Se il numero non è valido, viene lanciata un\textquotesingle{}eccezione di tipo Numero\+Non\+Valido\+Exception.
